
# Titan

Titan is a distributed graph database. It is an hadoop-friendly implementation of [TinkerPop]
[Blueprints]. Therefore it also use ThinkerPop REPL [Gremlin], and Front server [Rexster]

    module.exports =
      deps: {}
      configure:
        'ryba/titan/configure'
      commands:
        'prepare':
          'ryba/titan/prepare'
        'install': [
          'ryba/hbase/client'
          'ryba/titan/install'
          'ryba/titan/check'
        ]
        'check':
          'ryba/titan/check'
