
## Configure Rexster

*   `admin` (string | user object)
    The login name or a user object (see Nikita User documentation).
*   `config` (object)
    Object for Rexster configuration (xml) file

Example:

```json
{
  "ryba": {
    "rexster": {
      "admin": {
        "name": "rexster"
        "password": "rexster123"
      },
      "config": {
        "http": {
          [...]
        }
      }
    }
  }
}
```
